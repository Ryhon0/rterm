# Building
## Build dependencies
To build RTerm you will need the following dependencies:  
`gdc (gcc-d) dub`
## Dependencies
You need the following dependencies to run rterm:  
`gtk3 vte3 yajl`

## Compiling
You can build RTerm using `make build`
You can run it using `make run`

## Installing

### Linux, for current user (`~/.local`):
`make userinstall`  
To uninstall, run:  
`make useruninstall`
